# SCT_WD_3
# 🚀 Product Roadmap Development

This project presents a visually appealing and structured **Product Roadmap** for a new feature launch over a 3-month timeline. It is designed to demonstrate planning, prioritization, and progress tracking using modern web technologies.

---

## 📌 Project Overview

The roadmap helps in organizing product development by defining clear priorities and milestones. It uses the **MoSCoW prioritization technique** to categorize features and provides a timeline view for better planning and execution.

---

## ✨ Key Features

* 📊 MoSCoW Prioritization (Must, Should, Could, Won’t Have)
* 📅 3-Month Development Timeline
* 🎯 Milestone-based planning
* 📈 Status indicators (In Progress, Not Started, Upcoming)
* 🎨 Modern UI with glassmorphism design
* ⚡ Interactive button for user engagement
* 📱 Responsive layout for different devices

---

## 🛠️ Technologies Used

* HTML5
* CSS3 (Flexbox, Gradients, UI Styling)
* JavaScript (Basic Interactivity)

## 📊 MoSCoW Method Explained

* **Must Have** → Essential features required for launch
* **Should Have** → Important but not critical
* **Could Have** → Optional enhancements
* **Won’t Have** → Features excluded for now

---

* **Month 2** → Improvements and enhancements
* **Month 3** → Testing and final launch
